# Online Cheating Detection System

A web-based proctoring system that monitors students during online examinations using computer vision techniques to detect suspicious activities.

## Features

- Face Detection
- Head Movement Detection
- Real-Time Monitoring
- Violation Alerts

## Technologies Used

- Python
- Flask
- OpenCV
- HTML
- CSS
- JavaScript

## Installation
pip install -r requirements.txt

## Run
python app.py

Open:
http://127.0.0.1:5000
